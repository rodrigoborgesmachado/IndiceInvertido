Esse projeto abre com o codeblocks, caso tenha instalado basta abri-lo e executar que irá rodar normalmente.

Caso não tenha instalado, o arquivo main.c contém o código feito.
Caso queira visualizar os dados com exatidao, como nome dos arquivos lidos e de todas as palavras encontradas, basta descomentar algumas últimas linhas do código.

Caso queira ainda apenas executar o exe criado o caminho é esse: ..\bin\Debug\indiceInvertido.exe. Os arquivos de leitura já estão nessa pasta.
